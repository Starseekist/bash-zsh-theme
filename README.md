# bash-zsh-theme
A theme like bash for zsh

## feature

exactly as bash

## auto install:  

```bash
wget https://raw.githubusercontent.com/ycpedef/bash-zsh-theme/master/bash.zsh-theme -O $ZSH_CUSTOM/themes/bash.zsh-theme
omz theme use bash
```

## manually:  
just copy bash.zsh-theme to your zsh custom theme folder
